[h1]Building Themes 2[/h1]

Apply building themes to districts and cities. Only buildings in the selected themes will grow in a district. Subscribe to theme mods from the workshop, or create your own themes in-game.

[b]Building Themes 2[/b] is a community-maintained fork of the original [b]Building Themes[/b] by [url=https://github.com/boformer]boformer (Sebastian Schöner)[/url], updated to run on modern Cities: Skylines with Harmony 2.x. All original game logic and design belong to him.

[h1]Required Dependency[/h1]

[url=https://steamcommunity.com/sharedfiles/filedetails/?id=2040656402][b]► Harmony (Mod Dependency) — subscribe and enable this first[/b][/url]

The mod will silently do nothing if Harmony is not installed.

[h1]Compatibility[/h1]

[b]Compatible with Cities: Skylines 1.18.x[/b]

Tested against game version 1.18.1-f3.

This mod is [u]not[/u] compatible with:
[list]
[*][b]81 Tiles (original)[/b] — use the fixed version [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2862121823]81 Tiles 2[/url] if you need it
[*][b]Building Themes (original)[/b] — unsubscribe the original before using this fork, they cannot run together
[*][b]Building Simulation Overhaul[/b]
[*][b]Runways and Taxiways[/b] — use [url=http://steamcommunity.com/sharedfiles/filedetails/?id=465127441]Airport Roads[/url] instead
[/list]

Save games made with the original Building Themes mod are [b]fully compatible[/b] — your theme assignments will load normally.

[h1]What This Mod Does[/h1]
[list]
[*]Adds a [b]Themes[/b] tab to the district policies panel where you can select themes for a district or the whole city. The panel is unlocked from the start.
[*]Adds the [b]Building Theme Manager[/b] (accessible from the Themes tab) where you can edit existing themes and create your own.
[*]Provides two built-in themes: [b]European[/b] and [b]International[/b].
[*]Improves building footprint selection so themes with only small buildings (e.g. 1×2) work correctly.
[/list]

[h1]What This Mod Does Not Do[/h1]
[list]
[*]It will not corrupt save games if you disable it (cloned buildings will disappear, but the save is intact).
[*]It does not unlock European growables for non-European biomes. Use [b]European Buildings Unlocker[/b] for that.
[/list]

[h1]How To Use Themes[/h1]

Use the [b]Themes[/b] tab in the district policies panel to enable or disable themes for a district. You can select any number of active themes. If no theme is active, any growable building can spawn.

By default no theme is activated. If you also use [b]European Buildings Unlocker[/b] (recommended), both International and European buildings will spawn by default.

Districts inherit the city-wide theme setting. Check [b]Enable Theme Management for this district[/b] to control a district's themes separately.

If you redraw district borders, buildings outside the theme will not disappear. When they upgrade, they will level up to a building that fits the theme.

To allow custom workshop assets in a district, add them via the Theme Manager or check [b]Allow buildings which are not in any theme[/b].

[h1]Theme Manager[/h1]

Open the Theme Manager from the Themes tab.

The left panel shows all themes. Select a theme to add or remove buildings. A live rotatable 3D preview appears on the right.

Click a building in the list to select it. Two advanced options are available:
[list]
[*][b]Spawn rate[/b] — controls how often a building appears relative to others with the same zone type, level, and footprint.
[*][b]Upgrade building[/b] — forces a specific building when this one upgrades (e.g. same model with a pool and solar panels).
[/list]

[b]How the simulation works:[/b] A new zone spawns a level 1 building first (include one in your theme). It upgrades when requirements are met. The level 2 building must have the same width and equal or smaller length. Example: an L1 2×3 low-density residential upgrades to L2 2×3, 2×2, or 2×1.

If there is enough space, the game expands a building's front- or backyard to a maximum of 4 cells. This mod refines that: if no props or trees exist in the expanded area, the plot is shrunk back to the building's actual footprint. This allows tightly packed 4-cell-wide block layouts (as used by the UK Terraced Housing theme).

[h1]Asset Cloning[/h1]

The Theme Manager lets you clone buildings and assign different wealth levels to them — useful when you don't have enough assets for every level, or to add variety.

To clone: select a building, press [b]Clone Building[/b], enter a name, and choose the wealth level. The clone appears in the building list.

[u]Important:[/u] Clones are generated during level loading. Return to the main menu and reload your save after creating clones.

[u]Important:[/u] Deleting a theme that contains clones, or disabling the mod, will cause those buildings to disappear on next load. The save file itself is not broken.

Cloning can be disabled completely in the mod options.

[h1]How The Mod Saves Your Data[/h1]
[list]
[*]Custom themes and options are stored in [b]BuildingThemes.xml[/b] in your Cities: Skylines install folder.
[*]District theme assignments are stored inside each save game (key: [i]BuildingThemes-SaveData[/i]).
[/list]

You do not need to edit these files — everything is configurable in-game.
